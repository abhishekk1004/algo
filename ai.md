# AI Overview

This document lists the main topics in artificial intelligence with a short description and a simple example for each.

## 1. Machine Learning (ML)
Learns patterns from data to make predictions or decisions. Includes supervised, unsupervised, and reinforcement learning.
Example: Train a classifier to predict email spam (supervised).

## 2. Deep Learning (DL)
Uses multi-layer neural networks to learn hierarchical features from large datasets.
Example: Use a convolutional neural network to classify images of animals.

## 3. Natural Language Processing (NLP)
Enables machines to understand, generate, and transform human language.
Example: Build a sentiment analyzer that classifies reviews as positive or negative.

## 4. Computer Vision (CV)
Techniques for interpreting and processing images and videos.
Example: Detect faces in photos and blur them for privacy.

## 5. Reinforcement Learning (RL)
Learns to make sequences of decisions by trial and error to maximize rewards.
Example: Train an agent to play a video game using rewards for scoring points.

## 6. Robotics
Combines sensing, planning, and control for physical agents to interact with the real world.
Example: A delivery robot that plans paths and avoids obstacles.

## 7. Probabilistic Methods & Bayesian AI
Models uncertainty explicitly using probabilities and Bayesian inference.
Example: Use a Bayesian filter to track object location with noisy sensors.

## 8. Search and Optimization
Algorithms for exploring solution spaces or optimizing objective functions.
Example: Use A* search to find the shortest path on a grid map.

## 9. Expert Systems & Knowledge Representation
Rule-based systems that encode domain knowledge and reasoning.
Example: A medical diagnostic system that suggests possible conditions from symptoms.

## 10. Ethics, Safety, and Interpretability
Human-centered concerns about fairness, robustness, transparency, and alignment.
Example: Test a loan model for bias across demographic groups and add explainability reports.

## Quick learning path
- Start with Python and core ML libraries (`scikit-learn`, `pandas`, `numpy`).
- Learn supervised ML basics, then move to deep learning (`PyTorch`/`TensorFlow`).
- Practice with small projects: classification, regression, NLP, and CV tasks.
